gui.js

GUI движок для three.js. Позволяет создавать интерфейсы средствами WebGL через three.js.

Форму элемента задает свойство geometry.

Включает возможность создания следующих элементов:

  • GUI.Button - кнопка
  • GUI.Panel - панель
  • GUI.Label - метка
  • GUI.RadioGroup - элемент, группирующий радиокнопки
  • GUI.RadioButton - радиокнопка
  • GUI.CheckBox - флажок
  • GUI.EditBox - поле для ввода текста
  • GUI.ScrollBar - скроллбар
  • GUI.DialogBox - диалоговое окно

Для всех элементов имеются переопределяемые методы onmousedown(object), onmouseup(object), onmouseover(object), onmouseout(object), где object - подэлемент, для которого произошло событие
